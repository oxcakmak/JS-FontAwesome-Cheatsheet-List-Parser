# JS-FontAwesome-Cheatsheet-List-Parser
JS FontAwesome Cheatsheet List Parser

Go to "https://fontawesome.com/v4.7.0/cheatsheet/"<br>
Open the Developer Console.<br>
Paste the codes in "parser.js" to the console and the icon list will be sorted.<br><br>
